Export Markdown Files

Export blog entries as Markdown files for download in a ZIP file.
Information

Your blog entries will be exported as Markdown files in a format similar to the sample below. These files will be placed into a ZIP file and a download will start. This is useful as a means of a readable backup of your blog entries.

Please be patient as this could take a minute.

Title: Blog article title
Date: 2014-01-01 14:22
Author: Adam Presley
Tags: development, testing, news
Slug: blog-article-title

This is a blog entry. The header above is important because it will
be parsed to figure out the specifics of your entry. The rest is
just Markdown.
