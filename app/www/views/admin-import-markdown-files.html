Import Markdown Files

Import well-formed Markdown files as blog entries.
Instructions

Select one or more Markdown files to be uploaded and imported into your database as a blog entry. Each file must be formatted in a specific way. Below is an example of how they should look.

Title: Blog article title
Date: 2014-01-01 14:22
Author: Adam Presley
Tags: development, testing, news
Slug: blog-article-title

This is a blog entry. The header above is important because it will
be parsed to figure out the specifics of your entry. The rest is
just Markdown.
